# Commands
**?help** - Übersicht der Commands<br>
**?uptime** - wie lange ist der Bot schon online<br>
**?bot** - Kleine Rand information vom Bot.<br>

Music-CMDs<br>
**?play [URL]** - Spielt aktuellen Link ab<br>
**?queue** - Zeigt aktuelles Lied an<br>
